# 專案簡介

使用 React 框架 --> 遊戲介面  
基本狀態:顯示人物狀態  
背包:
添加物品可隨機獲得物品  
物品可裝備,出售  
屬性:顯示屬性數值  
探索:消耗體力 可獲取經驗.金幣
休息:消耗金幣 可完全回復體力.能量  
採礦:消耗體力 獲取金幣
強化:部位有穿戴裝置.可進行部位強化
